Off Leash


Previous Next

This is my front-end capstone project I made while attending Nashville Software School. I was excited to do this project because I wanted to work with the Google Maps API. In the app you're able to create an account, save favorite parks for future visits, add contacts you met at a park and post to social media outlets to let people know what park you're currently at. I used Firebase to save all of the data gathered on the app. Each individual account that is created is saved to Firebase and the contacts and favorites they add are linked to that particular account.
