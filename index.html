Off Leash


Previous Next

This is my front-end capstone project I made while attending Nashville Software School. We were to make a CRUD based app using AngularJS and Firebase. In the app you're able to create an account, save favorite parks for future visits, add contacts you met at a park and post to social media outlets to let people know what park you're currently at. I used Firebase to save all of the data gathered on the app. Each individual account that is created is saved to Firebase and the contacts and favorites they add are linked to that particular account.


Technologies used: AngularJS, Materialize, HTML/CSS, Firebase, Google Maps API
